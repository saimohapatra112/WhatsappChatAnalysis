# 📊 WhatsApp Chat Analyzer with Voice-Enabled Insights

Analyze your WhatsApp chat history with interactive visualizations and voice-enabled queries!

## 🔧 Features

- 📈 Message stats: total messages, words, media, and links
- 🕒 Timeline analysis: monthly and daily trends
- 👥 Top contributors in group chats
- 🧾 Common words and word clouds
- 😂 Emoji usage breakdown
- 🎤 **Voice-Enabled Insights** – Ask questions using your voice (e.g., "Who is most active?", "Show January activity")
